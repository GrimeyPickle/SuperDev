## SuperDev

A Browser Extension and a Swiss Army Toolbox for Web Designers, Developers and Founders.  
List of Feature below -

```
1. Session Manager
2. Color Picker
3. Color Palette
4. Text Editor
5. Lorem Ipsum
6. Take Screenshot
7. Extract Media
8. Delete Element
9. Responsive Viewer
10. Page Ruler
11. CSS Viewer
12. CSS Editor
13. Fake Form Filler
14. Go Incognito
15. Console Viewer
16. JSON Viewer
17. Clear All Cache
18. CORS Unblock
19. Cookie Editor
20. LocalStorage Editor
```
